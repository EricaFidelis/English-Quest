# English-Quest
Web-app interativo de flashcards para acelerar o vocabulário em inglês. Possui 250 palavras categorizadas, sistema de áudio nativo (TTS) para prática de pronúncia, desafio de 10s por rodada e recordes. Desenvolvido com HTML5, Tailwind CSS e JavaScript, focado em alta performance, responsividade mobile e interface moderna (UI/UX).

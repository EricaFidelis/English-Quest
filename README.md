#English Quest 
English Quest é um web-app gamificado projetado para acelerar a aquisição de vocabulário em inglês. Através de um sistema de flashcards de alta performance, o usuário é desafiado a associar palavras em tempo recorde, treinando tanto a leitura quanto a audição.

🚀 Funcionalidades Core
⚡ 250 Palavras Categorizadas: Banco de dados robusto dividido em: Food, Objects, Animals, School e Work.

🎧 Sistema de Áudio Nativo (TTS): Integração com a API de síntese de voz para prática imediata de pronúncia britânica.

🏆 Sistema de Gamificação:

Timer de 10s: Desafio de tempo para estimular o pensamento rápido.

Score & Highscore: Registro de pontuação em tempo real para estimular a superação.

Sistema de Vidas: 3 chances por rodada com feedback visual de erro (efeito shake).

📱 Interface Responsiva: Design focado em dispositivos móveis e desktop utilizando Glassmorphism e Tailwind CSS.

🛠️ Tecnologias Utilizadas
O projeto foi construído focando em ser leve e rápido, sem dependências pesadas:

HTML5: Estrutura semântica.

Tailwind CSS: Estilização utilitária e design responsivo.

JavaScript (ES6+): Lógica do jogo, manipulação de DOM e timers.

Web Speech API: Para a funcionalidade de conversão de texto em fala.

Lucide Icons: Para ícones vetoriais modernos.
